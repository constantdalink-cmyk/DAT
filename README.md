## 📖 这是什么？

**DAT**（Destroy And Transcend）是一款系统级镜像抹杀与重置工具。

支持在**原系统运行环境下**，无需 PE、无需 U 盘、无需外部引导、无需任何前置条件，直接对已知系统镜像进行**可选式抹杀与重置**。

> 简单来说：**不用 U 盘，不用 PE，不用重启到其他环境，直接在当前系统里干掉并重装系统。**

## ✨ 特性

- 🔥 **无前置条件** — 不依赖 PE / WinRE / 外部引导，原系统直接操作
- 🎯 **全系统可选** — 支持 Windows / Linux / macOS 等主流系统镜像
- 💀 **深度抹杀** — 系统级数据销毁，不留残余
- 🔄 **一键重置** — 自动识别已知镜像，智能部署
- 🛡️ **安全机制** — 多重确认 + 数据备份提醒，防止误操作
- 📦 **单文件分发** — 开箱即用，无需安装

## 🚀 快速开始

### 环境要求

- 操作系统：Windows 10+ / Linux (kernel 5.0+)
- 权限：管理员 / root
- 磁盘空间：≥ 2GB 可用

### 使用

```bash
# 查看帮助
dat --help

# 扫描当前系统已知镜像
dat scan

# 交互式抹杀并重置
dat reset --interactive

# 指定目标镜像重置
dat reset --target windows11 --image /path/to/win11.iso
```
